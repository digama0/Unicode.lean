def Lean.Parser.Term.byTactic' :
Lean.Parser.Parser

This is the same as byTactic, but it uses a different syntax kind. This is used by show and suffices instead of byTactic because these syntaxes don't support arbitrary terms where byTactic is accepted. Mathport uses this to e.g. safely find-replace by exact $e by $e in any context without causing incorrect syntax when the full expression is show $T by exact $e.
