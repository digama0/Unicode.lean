Execute x, save resulting expression and new state. We remove any Info created by x. The info nodes are committed when we execute applyResult. We use observing to implement overloaded notation and decls. We want to save Info nodes for the chosen alternative.

Auxiliary datatatype for presenting a Lean lvalue modifier. We represent a unelaborated lvalue as a Syntax (or Expr) and ListLVal. Example: a.foo[i].1 is represented as the Syntax a and the list [LVal.fieldName"foo",LVal.getOpi,LVal.fieldIdx1]. Recall that the notation a[i] is not just for accessing arrays in Lean.
